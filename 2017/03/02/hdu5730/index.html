Shell Necklace

有一个很显然的递推式：$dp[i]=\sum_{j=0}^{i-1} dp[j]a_{i-j}$，当然朴素的算法是$O(n^2)$，肯定过不了。观察这个式子，发现符合fft的形式（了解FFT），所以使用cdq分治+FFT（套路），复杂度为$O(nlog^{2}n)$。
